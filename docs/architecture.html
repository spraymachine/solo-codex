Core Modules

Auth & Space

Identity + Pair Model

Supabase Auth handles all identity. Each pair of users shares one Space. Two Persona slots (A/B) per space — user-named, accent-colored. Invite system: creator gets link/code, partner joins and claims slot B.

AuthGate AuthPanel (SSO + email) OAuthButton SetupWizard InviteAccept

Arc System

Time-boxed Goals

Core feature. Each Arc has user-set start/end dates, rank (S/A/B = difficulty 1/2/3), and a step checklist. 3–5 active arcs max per persona. Two progress bars per card: time elapsed + steps done. Partner-visible by default, hideable per arc.

ArcCard (dual bars) ArcForm ArcStepList ArcLimitBanner

Sync Engine

Local-first + Cloud

IndexedDB (Dexie) is the primary store. All writes hit Dexie first. CloudSync debounces 700ms then pushes to Supabase snapshot + individual tables. Realtime subscription pulls partner updates live. Offline queue in localStorage, flushed on reconnect.

Dexie (IndexedDB) CloudSync component solo_snapshots Realtime channel

Daily Record

Journal + Mood

Quick log entries (timestamped), reflection, gratitude list, and a pug mood sticker (great/good/okay/bad/could-do-better). Each item has a public/private toggle. Partner sees public items live. First log of day awards XP + updates streak.

MoodPicker (5 pug stickers) QuickLogInput ReflectionEditor GratitudeList

Spark + Mascot

Daily Message

One quote or message per persona per day, delivered by the pug mascot. Unique constraint enforces the daily limit. Mascot has idle state (no spark yet) and active state (spark waiting). Sent sparks lock for the day.

PugMascot (SVG/Lottie) SparkCompose SparkDisplay sparks table (unique/day)

Calendar Integration

V3 — External Sync

Phased delivery. 23a: ICS export + subscribe URL (no OAuth). 23b: Google Calendar direct push via OAuth2. 23c: Apple via CalDAV. 23d: Outlook via MS Graph. Tokens server-side only (edge functions), never in browser.

ICS generator Google Calendar API CalDAV client calendar_connections table

Privacy Model

Arcs

visibleToPartner: boolean
Toggle per arc. Default visible. Hidden arcs invisible in all partner views. Owner sees own hidden arcs with [hidden] badge.

Todos

visibleToPartner: boolean
Toggle per item (inline 👁 icon). Default visible. Partner sees count of hidden items: "[N] hidden today".

Journal

isPublic: boolean per entry, per gratitude item, per reflection.
Default public. Private items shown with 🔒 to owner only.

Mood

moodIsPublic: boolean
Default public. Partner sees your pug sticker for the day. Also shown as mini sticker on calendar cells.
